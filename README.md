# Turtlebot4 Automatic Driving with SLAM

본 프로젝트는 Turtlebot4를 사용하여 실내환경에서 위치추정이 가능한 RadioMap을 구축하였습니다.

## 1. 프로젝트 개요
- Turtlebot4를 활용하여 실내환경에서 SLAM을 수행하여 위치추정이 가능한 RadioMap 구축
- ROS2 Humble 기반으로 개발
- path_logger를 통해 로봇의 위치 데이터 수집
- Automatic Driving을 통해 자율탐사 진행
- SLAM을 통한 실시간 지도 생성

## 2. 주요 기능
- **SLAM (Simultaneous Localization and Mapping)**: 로봇이 이동하면서 주변 환경을 인식하고 지도를 생성하는 기술
- **Automatic Driving**: 로봇이 자율적으로 탐사하며 위치 데이터를 수집하는 기능
- **RadioMap 구축**: 수집된 위치 데이터를 기반으로 실내환경의 라디오 신호 강도를 시각화하여 RadioMap을 구축

## 3. 사용 방법
1. ROS2 Humble 설치 및 환경 설정
2. Turtlebot4 패키지 설치
